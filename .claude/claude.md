# 要件定義書（憲法）

用途：最終判断基準（解釈禁止）
配置：.claude/claude.md

## 0. 本憲法の位置づけ

本ドキュメントは、本プロジェクトにおける
**最上位ルール（例外・裁量・解釈を一切認めない）** である。

ディレクター・エンジニア・AI は、
すべての判断において本憲法を最優先とする。

## 1. プロジェクト思想（変更不可）

本プロジェクトは
**「売るサイト」ではなく「判断させるサイト」** を構築する。

短期CV最大化・商標訴求・ランキング最適化は
すべて **失敗状態** と定義する。

成功とは以下のみを指す。

- 商標ゼロ構成
- Google広告一般KWのみでの審査通過
- 制限・警告なしでの長期配信
- 案件差し替えが構造的に可能

## 2. 絶対前提（例外なし）

- 商標・ブランド・サービス名を一切使用しない
- 公式画像・ロゴ・公式感のあるUIは禁止
- ランキング・No.1・おすすめ表現は禁止
- 効果・価格の断定表現は禁止
- 指名検索・商標KW前提の設計は禁止

## 3. 判断優先順位（迷ったら必ず従う）

1. Google広告ポリシー違反リスク回避
2. 商標・ASPトラブル回避
3. 再利用・量産・差し替え耐性
4. CVR改善

※ ①〜③に抵触する施策は、CVRが向上しても即却下

## 4. 明確な失敗定義

以下が1つでも発生した場合、本プロジェクトは失敗とする。

- 商品名がないと成立しない構造
- 商標を入れた方が楽な設計
- 広告文とLP内容の乖離
- 案件差し替えにコピー変更が必要

## 5. 最終原則

迷ったら
**「出さない」「書かない」「止める」**
それが常に正解である。
